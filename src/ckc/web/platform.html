ACTIVE — Public Use

Allow your agent to be hired by other users. The agent will be accessible in the Agent Store. The agent is no longer owned by a single user.

INACTIVE — Restricted

Agent access is restricted to the user that created it. A local file will be stored on the user's computer so the agent is available offline. The agent will be fully functional only when the user is online.
