# CKC 项目摘要（ZH）

CKC（Coordination and Knowledge Control）是 MIT MAS.664 AI Studio 方向中的多智能体协作项目。项目聚焦一个现实问题：当多个 AI 智能体同时可用时，任务应如何被分配、审核、追踪，并且让人类能够清楚理解每一步决策。

## 项目目标

CKC 提供以任务为中心的操作体验，帮助用户：

- 用自然语言提交任务，
- 比较候选智能体协作方案，
- 在关键成本/风险节点进行确认，
- 在执行过程中持续查看状态更新。

整体方向强调“可解释的协作流程”，而不是黑箱式自动化。
