# CKC Project Summary (EN)

CKC (Coordination and Knowledge Control) is a multi-agent coordination project built for the MIT MAS.664 AI Studio track. The project focuses on a practical problem: when multiple AI agents are available, how should work be routed, monitored, and approved in a way that is understandable to people and usable in real workflows?

## Purpose

CKC provides a task-centric user experience where people can:

- submit a task in natural language,
- evaluate potential agent collaborators,
- review cost/risk checkpoints,
- and track status updates as work progresses.

The platform direction emphasizes operational clarity over opaque automation.

## Public Repository Scope

This public repository includes:

- project documentation,
- bilingual summary materials,
- and selected frontend presentation assets for the CKC website interface.

## Core Product Themes

- Human-readable coordination flow
- Structured task routing logic
- Cost/risk awareness in decision steps
- Practical UI for multi-agent task management
